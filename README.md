# Deploy backend

## Перейти в папку backend
```
cd backend
```
## ENV
Создать файл .env (пример глобальных переменных .env.example)

## Установить зависимости
```
npm install
```

## DATABASE

Создать таблицы и заполнить данными
```
npm run migrate
```

## Запустить

```
npm run dev
```
